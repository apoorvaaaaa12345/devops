1. **Git** → Tracks and manages changes in source code using version control.

2. **GitHub** → Online platform to store, share, and collaborate on Git repositories.

3. **Flask** → Lightweight Python framework used to build web applications and APIs.

4. **Jenkins** → Automation server used for Continuous Integration and Continuous Deployment (CI/CD).

5. **Ansible** → Automation tool for server configuration, deployment, and management.

6. **Gradle** → Build automation tool used to compile, test, and package applications.

7. **Selenium** → Automation testing tool used to test web applications in browsers.

8. **Docker** → Platform that packages applications into portable containers.

9. **Kubernetes** → Container orchestration platform used to deploy and manage containers at scale.

10. **Snyk** → Security tool that scans code and dependencies for vulnerabilities.

11. **Maven** → Java project management and build tool that handles dependencies and builds.

12. **ChromeDriver** → Browser driver that allows Selenium to control Google Chrome.

13. **Minikube** → Tool that runs a local Kubernetes cluster on your computer.

14. **kubectl** → Command-line tool used to interact with Kubernetes clusters.
